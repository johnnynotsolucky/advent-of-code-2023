# Advent of Code 2023

My solutions for [Advent of Code 2023](https://adventofcode.com/2023) written in rust.

## Tests

```
cargo test
```

## Run the solution for a day (both parts)

```bash
cargo run --release -- <day> <part>
```

```bash
cargo run --release -- 5 2
```
